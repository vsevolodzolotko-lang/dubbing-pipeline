# tests/

Тестові вхідні файли для перевірки пайплайну: аудіо-фрагменти, текстові транскрипти, очікувані результати.

Використовуються для ручного smoke-тесту та інтеграційних перевірок перед повноцінним batch-запуском.

Аудіофайли (`.mp3`, `.wav`, `.m4a`) виключені з репозиторію через `.gitignore` — зберігай їх локально або на спільному диску.
